#lesson14:命令行
cd nodejsTest:进入nodejsTest文件夹;
dir: 查看文件夹下所有文件；
md lesson14: 创建lesson14文件夹；
cd le(文件夹字母)+tab键: 补全文件名称；
cd.>index.js: 创建index.js文件；
cd ..: 返回上级目录；
cd .: 当前目录；
rd del: 删除del文件夹;
cls: 清屏；
exit: 退出命令行

#lesson17:运行node文件
命令行方式运行：cmd下执行文件(1.cmd窗口下输入文件路径找到文件；2.文件夹下找到文件，在当前文件目录下输入cmd,回车；)
执行命令：node xxx(文件名称)
